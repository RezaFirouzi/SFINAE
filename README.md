# SFINAE

This repository contains example source codes describing SFINAE is C++ template (meta) programming.

SFINAE : Substitution Failure Is Not An Error

The examples illustrates what happen under the hood in:

- std::is_same
- std::is_same_v
- std::is_same_t
- std::enbale_if
- std::enable_if_t
